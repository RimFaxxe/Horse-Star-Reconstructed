This folder contains the minimal versions of the missing XMLs that have been reconstructed based on recovered data and findings from the Assembly-CSharp.dll file in the Horse Star directory. These are not the original XML files generated by the game - they're minimal versions rewritten for testing purposes for this project. The game accepts them and asks for EN_Dialog.txt and Input_DefinitionsQWERTY.txt. These XML files are not intended for public use and are only meant to be used for this project. These files exist for testing purposes only and are in no way finalized. They are uploaded as part of the research and attempt to recontruct Horse Star.
